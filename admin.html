Экспорт и публикация

После редактирования товаров и настроек нужно скачать обновлённые файлы и загрузить их на GitHub, чтобы изменения появились на сайте.

  1. Нажмите кнопку «Скачать products.js» — скачается файл с товарами
  2. Нажмите кнопку «Скачать config.js» — скачается файл с настройками
  3. Откройте папку сайта на компьютере (ваш отдых/js/)
  4. Замените файлы products.js и config.js в папке js/
  5. Сделайте git add . && git commit -m "update products" && git push
  6. Через 1–2 минуты сайт обновится автоматически
